## Нерешённые пункты

### 🔄 Оффлайн-тайлы карты (п. 3.7)
Для полностью оффлайн-карты необходима **платная лицензия Яндекс MapKit** с поддержкой кэширования тайлов. Без неё:
- Карта загружает тайлы из интернета
- Все остальные функции (координаты, данные захоронений, синхронизация) — полностью оффлайн

### ✅ Поиск места по номеру участка (п. 5.2) — выполнено
Реализована постоянная строка поиска в шапке карты. Поиск по участку, ряду, номеру места или полному номеру (например «5-3-10»). При выборе результата камера центрируется на месте и выделяет его.
